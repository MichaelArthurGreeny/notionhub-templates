# NotionHub Templates - QuickStart (5 minut)

## Najszybsza droga do uruchomienia

### Krok 1: Wgraj na GitHub (2 min)

1. Wejdź na https://github.com i zaloguj się
2. Kliknij zielony przycisk "New" (nowe repozytorium)
3. Nazwa: `notionhub-templates`
4. Kliknij "Create repository"
5. Przeciągnij cały folder `notionhub-templates` do okna GitHub
6. Kliknij "Commit changes"

### Krok 2: Deploy na Vercel (2 min)

1. Wejdź na https://vercel.com
2. Kliknij "Sign up" → "Continue with GitHub"
3. Kliknij "New Project"
4. Znajdź `notionhub-templates` i kliknij "Import"
5. Kliknij "Deploy"
6. Poczekaj 1-2 minuty...
7. **Gotowe!** Kliknij link żeby zobaczyć swoją stronę

### Krok 3: Dodaj produkty do Gumroad (później)

1. Załóż konto na https://gumroad.com
2. Dodaj produkty według `notion-templates/README.md`
3. Zaktualizuj linki w `src/data/templates.ts`
4. Wgraj zmiany na GitHub (Vercel auto-redeploy)

## Struktura plików

```
notionhub-templates/
├── src/
│   ├── app/           # Strony Next.js
│   ├── components/    # Komponenty React
│   └── data/          # Dane szablonów
├── notion-templates/  # Dokumentacja szablonów
├── INSTRUKCJA.md      # Pełna instrukcja
├── QUICKSTART.md      # Ten plik
├── deploy.bat         # Skrypt Windows
└── deploy.sh          # Skrypt Mac/Linux
```
